# React Native CLI Project

A React Native project created using the React Native CLI.



1. Clone the repository:
```bash
git clone <your-repository-url>
cd fresh
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

## Running the App

### For Android:
```bash
npm run android
# or
yarn android
```

### For iOS:
```bash
npm run ios
# or
yarn ios
```
